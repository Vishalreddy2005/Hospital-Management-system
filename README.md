# Hospital Management System

A comprehensive Java-based application designed to streamline hospital operations, manage patient records, and coordinate department activities efficiently.

## 🚀 Features
* **User Authentication:** Secure login system for administrators and staff.
* **Patient Management:** Add, update, and view patient details and medical history.
* **Reception Desk:** Manage appointments and initial check-ins.
* **Department Overview:** Track different hospital departments and their status.
* **Database Integration:** Persistent storage for all hospital data.

## 🛠️ Tech Stack
* **Language:** Java
* **Database:** MySQL (JDBC)
* **GUI Framework:** Java Swing / AWT

## 📋 Prerequisites
Before running the project, ensure you have the following installed:
* Java Development Kit (JDK) 8 or higher
* MySQL Server
* A Java IDE (IntelliJ IDEA, Eclipse, or NetBeans)
